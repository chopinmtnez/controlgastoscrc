🏦 ING — Importación de cobros
Alternativa a la API Open Banking

La API Open Banking de ING bajo PSD2 requiere registrarse como TPP (Third Party Provider) ante el Banco de España: licencia AISP, certificado eIDAS (~200€/año) y compliance legal. No es viable para uso personal.

La alternativa práctica: exportar los movimientos desde la app ING como CSV o Excel y subirlos aquí.

Cómo exportar desde ING

  1. Abre la app ING o web (ing.es)
  2. Ve a tu cuenta corriente
  3. Movimientos → Exportar / Descargar
  4. Elige formato CSV o Excel
  5. Sube el archivo aquí ↓
